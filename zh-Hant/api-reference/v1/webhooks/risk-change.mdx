---
title: "風險變動通知"
description: "當監控地址的風險狀態發生變化時觸發的 Webhook 通知。"
---

當您監控的地址由於各種原因（如被標記為高風險、關聯黑名單等）導致其風險狀態發生改變時，Arisk 會傳送此通知。

### 推送型別
`type`: `risk_change`

### 請求引數

| 引數名 | 型別 | 說明 |
| :--- | :--- | :--- |
| `type` | `string` | 推送型別，固定為 `risk_change` |
| `timestamp` | `number` | 推送時的 Unix 時間戳（秒） |
| `alert` | `object` | 告警詳情物件 |
| `alert.chain` | `string` | 鏈型別（如：`bsc`, `ethereum`, `tron`） |
| `alert.token` | `string` | 代幣符號（如：`usdt`, `usdc`） |
| `alert.monitoring_address` | `string` | 被監控的地址 |
| `alert.details` | `array` | 風險詳情列表 |
| `alert.details[].risk_level` | `number` | 風險等級：`0`-低, `1`-中低, `2`-中, `3`-中高, `4`-高 |
| `alert.details[].risk_type` | `array` | 風險型別標籤陣列 |

### 請求體示例

```json
{
  "type": "risk_change",
  "timestamp": 1767513871,
  "alert": {
    "chain": "bsc",
    "token": "usdt",
    "monitoring_address": "0xa49195f82885ab741aec6ab62e7e4f6dc1f47ece",
    "details": [
      {
        "risk_level": 4,
        "risk_type": ["black_industry"]
      }
    ]
  }
}
```
