## 版本歷史

### 2025 年

<AccordionGroup>
  <Accordion title="v2.0.0 - 2025年12月30日" icon="rocket">
    **主要功能:**
    - 🚀 **Arisk V2 API 正式釋出**：全面採用層次化 RESTful 架構，介面語義更清晰。
    - 📁 **深度整合批次管理**：支援地址批次新增與刪除，效能顯著提升。
    - 🧠 **全新分析任務引擎**：升級非同步處理機制，支援更深入的風險權重分解。
    - 🌐 **全方位國際化**：API 文件實現中英雙語 100% 覆蓋。
    
    **改進:**
    - 統一 OpenAPI 生產環境介面配置。
    - 最佳化文件庫導航結構，完善功能場景分類。
  </Accordion>
</AccordionGroup>

### 2024 年

<AccordionGroup>
  <Accordion title="v1.0.0 - 初始發行版" icon="star">
    **功能概覽:**
    - 🌍 **全鏈監控支援**：奠定了比特幣、以太坊等主流公鏈地址監控的基礎。
    - 🔔 **實時風險警報**：提供基礎的鏈上異常行為識別與警報通知。
    - 📊 **地址管理基礎**：支援監控地址的列表化管理與查詢。
  </Accordion>
</AccordionGroup>
