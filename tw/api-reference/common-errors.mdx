---
title: "通用錯誤回饋"
description: "在調用 API 時可能遇到的基礎系統錯誤說明。"
---

我们推测在使用的过程中，您可能会遇到以下错误回馈。您可以参考下述信息帮助您定位问题。

<ResponseField name="399101" type="Internal">
  Internal error (內部錯誤)
</ResponseField>

<ResponseField name="399102" type="Unavailable">
  Upstream service unavailable (上游服務不可用)
</ResponseField>

<ResponseField name="399201" type="Unauthenticated">
  Missing authorization header (缺少授權請求頭)
</ResponseField>

<ResponseField name="399202" type="Unauthenticated">
  Invalid authorization header format (授權請求頭格式錯誤)
</ResponseField>

<ResponseField name="399207" type="InvalidArgument">
  Unsupported chain (不支持的鏈)
</ResponseField>

<ResponseField name="399208" type="InvalidArgument">
  Unsupported token (不支持的代幣)
</ResponseField>

<ResponseField name="399209" type="InvalidArgument">
  Unsupported chain or token (不支持的鏈或代幣組合)
</ResponseField>
