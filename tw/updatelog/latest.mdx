# 版本 2.0.0
**釋出日期: 2025年12月30日**

<Note>
Arisk 正式進入 V2 時代！本次更新帶來了全新的 API 架構演進，我們在介面效能、管理便捷性及分析深度上實現了質的飛躍。
</Note>

## 🚀 Arisk V2 API 全新發布

V2 版本不僅是簡單的版本號迭代，更是我們對區塊鏈風控與地址監控能力的重構。

### 1. 深度 RESTful 架構演進
V2 API 採用資源導向設計，全面引入路徑引數（Path Parameters），使介面語義更加清晰直觀。相比 V1 的扁平化路徑，V2 的層次化結構讓開發者能更自然地操作特定資源，同時也提升了 API 的可擴充套件性與標準化程度。

### 2. 批次操作效能質變
針對企業級大規模地址管理場景，V2 引入了全新的批次操作介面：
- **批次新增**：支援單次請求新增多個監控地址，大幅減少 API 呼叫頻率。
- **批次管理**：透過統一的刪除介面高效清理監控列表，簡化維護工作量。
這些改進顯著降低了大規模整合時的網路往返次數與系統處理時耗。

### 3. 全新非同步分析任務系統
我們將原有的查詢功能全面升級為“分析任務（Analysis Task）”引擎：
- **更強的非同步處理能力**：支援更復雜、更高併發的分析請求，任務狀態查詢更加穩定可靠。
- **更深度的評估維度**：V2 提供了比 V1 更詳盡的風險權重分解與鏈上行為畫像，幫助您更精準地識別潛在風險。
- **任務鏈路追蹤**：每個分析任務都有唯一的任務 ID，方便您進行歷史回溯、審計與結果持久化。

### 4. 精細化監控配置
V2 最佳化了地址監控的狀態管理邏輯，支援更靈活的監控開關切換（如：啟用/暫停）。同時，我們最佳化了警報過濾機制，確保警報通知更加精準。

### 5. 國際化體驗升級
- **全方位文件覆蓋**：完成了 V1 與 V2 API 的中英文件全面覆蓋，填補了英文文件空白。
- **配置標準化**：OpenAPI 定義檔案中現已包含官方生產地址 `https://api.arisk.io`，支援更便捷的線上除錯。
