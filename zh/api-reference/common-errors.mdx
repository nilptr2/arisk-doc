### 响应结构

当 API 调用发生错误时，将返回包含以下字段的 JSON 响应：

- `code`: 6 位数字的业务错误码。
- `message`: 错误的具体描述信息。

示例：
```json
{
  "code": "104001",
  "message": "Invalid request parameters"
}
```


### 错误码格式

业务错误码的第一位数字代表错误的级别或类型：

- **1**: 参数或输入错误。
- **2**: 业务逻辑错误。
- **3**: 系统或内部错误。
