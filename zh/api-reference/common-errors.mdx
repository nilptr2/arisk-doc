---
title: "通用错误反馈"
description: "在调用 API 时可能遇到的基础系统错误说明。"
---

我们推测在使用 API 的过程中，您可能会遇到以下错误反馈。您可以参考下述信息帮助您定位问题并了解处理建议。

<ResponseField name="399101" type="Internal">
  Internal error (内部错误)
</ResponseField>

<ResponseField name="399102" type="Unavailable">
  Upstream service unavailable (上游服务不可用)
</ResponseField>

<ResponseField name="399201" type="Unauthenticated">
  Missing authorization header (缺少授权请求头)
</ResponseField>

<ResponseField name="399202" type="Unauthenticated">
  Invalid authorization header format (授权请求头格式错误)
</ResponseField>

<ResponseField name="399207" type="InvalidArgument">
  Unsupported chain (不支持的链)
</ResponseField>

<ResponseField name="399208" type="InvalidArgument">
  Unsupported token (不支持的代币)
</ResponseField>

<ResponseField name="399209" type="InvalidArgument">
  Unsupported chain or token (不支持的链或代币组合)
</ResponseField>
