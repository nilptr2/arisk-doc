### 字段说明

<ParamField body="metrics" type="string[]" optional>
  分析指标列表。若不传该字段或传空数组，则默认返回所有支持的指标。
  
  **校验规则**:
  *   传入值必须在支持的指标列表内，否则返回 `400 Invalid Params` 错误。
  *   数组中若有重复项会自动去重，且响应结果将尽量保持请求中的指标顺序。

  <Expandable title="支持的指标枚举值">
    <ResponseField name="overview" type="Metric">
      **地址总览** - 包含余额、地址类型（交易所/合约等）、首次转账时间及对手方、封禁/制裁状态。
    </ResponseField>
    <ResponseField name="txflow" type="Metric">
      **资金流向** - 包含资金流出的拓扑图数据（节点 nodes 与连线 links）。
    </ResponseField>
    <ResponseField name="habit" type="Metric">
      **交易习惯** - 包含按小时（0-23）分布的流入/流出金额与笔数统计。
    </ResponseField>
    <ResponseField name="historical_balance" type="Metric">
      **历史余额** - 包含每日历史余额变动曲线数据。
    </ResponseField>
    <ResponseField name="risk_type" type="Metric">
      **风险类型统计** - 包含按风险标签类型（如 Scam, Hacker, Mixer 等）分类的交易汇总。
    </ResponseField>
    <ResponseField name="top_transaction" type="Metric">
      **Top 交易对手** - 包含交易金额或频率最高的对手方地址列表。
    </ResponseField>
    <ResponseField name="risk_address_total" type="Metric">
      **风险地址汇总** - 包含涉及已知风险地址的交易金额、笔数及地址总数合计。
    </ResponseField>
  </Expandable>
</ParamField>
