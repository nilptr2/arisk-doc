---
title: "交易检测"
description: "当监控地址发生交易或发现风险交易时触发的 Webhook 通知。"
---

Arisk 实时监测监控地址的交易活动。当监控地址发生流入/流出交易，或与已知风险地址产生交互时，会推送此告警。

### 推送类型
支持以下三种类型：
- `transaction_risk`: 发现风险交易。
- `transaction_in`: 发现普通流入交易。
- `transaction_out`: 发现普通流出交易。

### 请求参数

| 参数名 | 类型 | 说明 |
| :--- | :--- | :--- |
| `type` | `string` | 推送类型：`transaction_risk`, `transaction_in`, `transaction_out` |
| `timestamp` | `number` | 推送时的 Unix 时间戳（秒） |
| `alert` | `object` | 告警详情对象 |
| `alert.id` | `string` | 告警唯一标识 ID |
| `alert.chain` | `string` | 链类型 |
| `alert.token` | `string` | 代币符号 |
| `alert.address` | `string` | 被监控的地址 |
| `alert.counterparty` | `string` | 交易对手方地址 |
| `alert.tx_hash` | `string` | 交易哈希 |
| `alert.amount` | `string` | 交易金额（字符串形式） |
| `alert.direction` | `number` | `0`: 流入监控地址, `1`: 流出监控地址 |
| `alert.block_number` | `number` | 区块高度 |
| `alert.block_time` | `string` | 区块时间（ISO 8601 格式） |
| `alert.risk_types` | `array` | 对手方风险类型。**注意**：当类型为 `transaction_in` 或 `transaction_out` 时，此字段为空数组。 |
