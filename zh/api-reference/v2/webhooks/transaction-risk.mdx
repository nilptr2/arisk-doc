---
title: "风险交易检测"
description: "当监控地址与已知风险地址发生交互时触发的 Webhook 通知。"
---

当监控地址与已知的风险地址产生交易往来时，Arisk 会实时推送此告警。

### 推送类型
`type`: `transaction_risk`

### 请求参数

| 参数名 | 类型 | 说明 |
| :--- | :--- | :--- |
| `type` | `string` | 推送类型，固定为 `transaction_risk` |
| `timestamp` | `number` | 推送时的 Unix 时间戳（秒） |
| `alert` | `object` | 告警详情对象 |
| `alert.id` | `string` | 告警唯一标识 ID |
| `alert.chain` | `string` | 链类型 |
| `alert.token` | `string` | 代币符号 |
| `alert.address` | `string` | 被监控的地址 |
| `alert.counterparty` | `string` | 交易对手方地址 |
| `alert.tx_hash` | `string` | 交易哈希 |
| `alert.amount` | `string` | 交易金额（字符串形式） |
| `alert.direction` | `number` | `0`: 流入监控地址, `1`: 流出监控地址 |
| `alert.block_number` | `number` | **[新增]** 区块高度 |
| `alert.block_time` | `string` | **[新增]** 区块时间（ISO 8601 格式） |
| `alert.risk_types` | `array` | 对手方所属的风险类型标签数组 |

### 请求体示例

```json
{
  "type": "transaction_risk",
  "timestamp": 1767514410,
  "alert": {
    "id": "019b8811-8e96-750c-9009-c195386aa19a",
    "chain": "tron",
    "token": "usdt",
    "address": "TUjKn1jon1t3V1cM8hKNg9ERxswXipQ2KG",
    "tx_hash": "00000000000000000000000000000000000000000000000000000000001e8481",
    "counterparty": "TWWCH84pZJgxXtrVqbjm3614gnuWp2RGHD",
    "amount": "500000000",
    "direction": 0,
    "block_number": 2000,
    "block_time": "2026-01-04T08:13:30Z",
    "risk_types": ["black_industry"]
  }
}
```
