---
title: "风险变动通知"
description: "当监控地址的风险状态发生变化时触发的 Webhook 通知。"
---

当您监控的地址由于各种原因（如被标记为高风险、关联黑名单等）导致其风险状态发生改变时，Arisk 会发送此通知。

### 推送类型
`type`: `risk_change`

### 请求参数

| 参数名 | 类型 | 说明 |
| :--- | :--- | :--- |
| `type` | `string` | 推送类型，固定为 `risk_change` |
| `timestamp` | `number` | 推送时的 Unix 时间戳（秒） |
| `alert` | `object` | 告警详情对象 |
| `alert.chain` | `string` | 链类型（如：`bsc`, `ethereum`, `tron`） |
| `alert.token` | `string` | 代币符号（如：`usdt`, `usdc`） |
| `alert.monitoring_address` | `string` | 被监控的地址 |
| `alert.details` | `array` | 风险详情列表 |
| `alert.details[].risk_level` | `number` | 风险等级：`0`-低, `1`-中低, `2`-中, `3`-中高, `4`-高 |
| `alert.details[].risk_type` | `array` | 风险类型标签数组 |

### 请求体示例

```json
{
  "type": "risk_change",
  "timestamp": 1767513871,
  "alert": {
    "chain": "bsc",
    "token": "usdt",
    "monitoring_address": "0xa49195f82885ab741aec6ab62e7e4f6dc1f47ece",
    "details": [
      {
        "risk_level": 4,
        "risk_type": ["black_industry"]
      }
    ]
  }
}
```
