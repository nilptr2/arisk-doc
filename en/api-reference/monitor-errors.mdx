### Response Structure

When an API call results in an error, a JSON response containing the following fields will be returned:

- `code`: A 6-digit business error code.
- `message`: A descriptive message for the error.

Example:
```json
{
  "code": "104001",
  "message": "Invalid request parameters"
}
```


### Error Code Format

The first digit of the business error code represents the category or type of error:

- **1**: Parameter or Input Error.
- **2**: Business Logic Error.
- **3**: System or Internal Error.
