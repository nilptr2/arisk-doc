---
title: "Transaction Risk Detection"
description: "Webhook notification triggered when a monitored address interacts with a known risk address."
---

This alert is pushed in real-time when a monitored address interacts with a known risk address.

### Push Type
`type`: `transaction_risk`

### Request Parameters

| Parameter | Type | Description |
| :--- | :--- | :--- |
| `type` | `string` | Push type, fixed as `transaction_risk` |
| `timestamp` | `number` | Unix timestamp (seconds) when pushed |
| `alert` | `object` | Alert details object |
| `alert.id` | `string` | Unique alert ID |
| `alert.chain` | `string` | Chain type |
| `alert.token` | `string` | Token symbol |
| `alert.address` | `string` | The monitored address |
| `alert.counterparty` | `string` | Counterparty address |
| `alert.tx_hash` | `string` | Transaction hash |
| `alert.amount` | `string` | Transaction amount (string format) |
| `alert.direction` | `number` | `0`: Inflow to monitored address, `1`: Outflow from monitored address |
| `alert.block_number` | `number` | **[New]** Block height |
| `alert.block_time` | `string` | **[New]** Block time (ISO 8601 format) |
| `alert.risk_types` | `array` | Array of risk type labels for the counterparty |
