## ğŸ”„ Migration Guide

### Breaking Changes

<Warning>
This release includes minor breaking changes for users of the beta webhook feature.
</Warning>

**Webhook Signature Verification** (Required for enhanced security):

```javascript
// Before (v2.4.x)
app.post('/webhook', (req, res) => {
  const data = req.body;
  // Process without verification
});

// After (v2.5.0)
const crypto = require('crypto');

app.post('/webhook', (req, res) => {
  const signature = req.headers['x-arisk-signature'];
  const payload = JSON.stringify(req.body);
  
  const expectedSignature = crypto
    .createHmac('sha256', WEBHOOK_SECRET)
    .update(payload)
    .digest('hex');
  
  if (signature !== expectedSignature) {
    return res.status(401).send('Invalid signature');
  }
  
  // Process verified webhook
});
```

<Steps>
  <Step title="Get Your Webhook Secret">
    Navigate to **Dashboard > Webhooks** and copy your webhook secret
  </Step>
  
  <Step title="Update Code">
    Add signature verification to your webhook endpoint
  </Step>
  
  <Step title="Test">
    Use the "Test Webhook" button in dashboard to verify
  </Step>
  
  <Step title="Deploy">
    Deploy your updated code - old webhooks will continue to work until Jan 2025
  </Step>
</Steps>
